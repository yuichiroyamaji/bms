# Infrastructure

AWS CDK infrastructure for deploying the Next.js admin dashboard to AWS AppRunner.

## Quick Start

```bash
# Install dependencies
npm install

# Bootstrap CDK (first time only)
npx cdk bootstrap

# Configure
# Edit config/app-config.ts with your settings

# Deploy to development
npm run deploy:dev

# Deploy to production
npm run deploy:prod
```

## Available Commands

```bash
npm run build           # Compile TypeScript
npm run watch          # Watch mode
npm run test           # Run tests

# Development
npm run deploy:dev     # Deploy dev environment
npm run diff:dev       # Preview dev changes
npm run synth:dev      # Generate dev template
npm run destroy:dev    # Delete dev stack

# Production
npm run deploy:prod    # Deploy prod environment
npm run diff:prod      # Preview prod changes
npm run synth:prod     # Generate prod template
npm run destroy:prod   # Delete prod stack
```

## Configuration

Edit `config/app-config.ts` to configure:
- Environment variables
- Instance sizing (CPU/Memory)
- Custom domain
- Alarm email

## Architecture

```
Developer → CDK → ECR → AppRunner → CloudWatch
                                  ↓
                                 SNS (Alarms)
```

## Cost

- Base: ~$3-5 USD/month (1 vCPU + 2 GB)
- Scales with traffic

## Monitoring

- CloudWatch Logs: `/aws/apprunner/nextjs-admin-dashboard`
- Alarms: CPU, Memory, HTTP errors
- Email notifications via SNS
